Demonstrating the background-size plugin for jQuery

Below is a small demonstration of the background-size plugin for jQuery.

This plugin is available as two shorthand functions that don't receive any parameters. See the full plugin call after the examples.

$('.element').cover()
$('.element').contain()

The examples

background-size: cover

with a portrait oriented image

background-size: cover

with a landscape oriented image

background-size: contain

with a portrait oriented image

background-size: contain

with a landscape oriented image

The background-size plugin

This demo uses the actual plugin which is able to force the execution of the plugin on non IE (below 9) browsers. The actual plugin can receive two parameters:

$('.element').background({ force: true, size: 'cover' });

The actual code from demo.js to render the examples below:

(function(window, $) {
   $('.cover').background({
       force: true,
       size: 'cover'
   });
   $('.contain').background({
       force: true,
       size: 'contain'
   });
})(window, jQuery);
