Demonstrating the background-size plugin for jQuery

Below is a small demonstration of the background-size plugin for jQuery.

This plugin is available as two shorthand functions that don't receive any parameters. For more control over the output see the full plugin call after the examples.

$('.element').cover()
$('.element').contain()

The examples

background-size: cover

with a portrait oriented image

background-size: cover

with a landscape oriented image

background-size: contain

with a portrait oriented image

background-size: contain

with a landscape oriented image

The background-size plugin

This demo uses the actual plugin which is able to force the execution of the plugin on non IE (below 9) browsers. The actual plugin can receive five parameters:

$('.element').background({
   force: true,
   size: 'cover',
   image: './path-to/image.jpg',
   position: 'left top',
   repeat: 'no-repeat'
});

Parameters

How to use the background plugin without the shorthand functions.

Parameter Values Default Description
force (boolean) true or false false Set to true to force the execution of the plugin on browsers with native background-size support
size (string) cover or contain cover The type of background sizing
image (string) url null A path to a image used as background image. When empty it uses the image defined in the stylesheet. If both are empty the plugin will throw an error.
position (string) px or % or string null The position of the background in CSS syntax (for example: 'left top'). When empty it uses the position defined in the stylesheet. If both are empty it defaults to '50% 50%'.
repeat (string) repeat or no-repeat null Whether to repeat the background (only in the case of size: 'contain'). When empty it uses the setting defined in the stylesheet. If both are empty it defaults to 'no-repeat'.

The demo code

The actual code from demo.js to render the examples below:

(function(window, $) {
   $('.cover').background({
       force: true,
       size: 'cover'
   });
   $('.contain').background({
       force: true,
       size: 'contain'
   });
})(window, jQuery);
